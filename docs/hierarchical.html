Hierarchical data structures

DataFrame can represent hierarchical data structures using two special types of columns:

  • ColumnGroup is a group of columns

  • FrameColumn is a column of dataframes

You can read DataFrame from json or from in-memory object graph preserving original tree structure.

Hierarchical columns can also appear as a result of some modification operations:

  • group produces ColumnGroup

  • groupBy produces FrameColumn

  • pivot may produce FrameColumn

  • split of FrameColumn will produce several ColumnGroups

  • implode converts ColumnGroup into FrameColumn

  • explode converts FrameColumn into ColumnGroup

  • merge converts ColumnGroups into FrameColumn

  • etc.
